## 技术笔记
* 全局安装 [vue-cli](https://cli.vuejs.org/zh/) Vue.js 开发的标准工具
  * npm install -g @vue/cli
  * 查看版本号：vue --version
* 创建一个基于webpack模板的新项目
  * vue create resources-nav-home
* 启动项目
  * cd resources-nav-home
  * npm run serve
* 使用图形化界面
  * vue ui
* 安装依赖 axios、vue-router、vuex
